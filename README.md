
# 🚴 Rider App  
### Smart Companion App for the Intelligent Helmet Ecosystem

The **Rider App** is a mobile application built to complement the smart helmet hardware system. It acts as the central interface for riders—bringing together navigation, communication, safety insights, and real-time ride data.

⚠️ **Status: Active Development**  
This project is currently under heavy development and is **far from complete**. Features are experimental, APIs may change, and stability is not guaranteed.

---

## 📌 Overview

The Rider App is designed to:

- 📍 Provide **navigation optimized for riders**
- 🔗 Connect with the smart helmet via **Bluetooth / ESP communication**
- 🚨 Display **crash alerts and safety data**
- 👥 Enable **group ride coordination (planned)**
- 🎧 Control **audio & ANC settings**
